Semana 3 - Exercício Prático

Nesse trabalho, vamos montar uma ideia de layout de site que integre alguns conjuntos de elementos (header, barra lateral e conteúdo central). Encaminho abaixo três exemplos que trazem algo parecido com a proposta para este trabalho. Para montar isso, vocês devem utilizar os elementos semânticos do HTML (header, aside, main, footer). Não se preocupe, o objetivo aqui é exercitar e não cumprir um layout perfeito. Faça o que você conseguir, quero que vocês entreguem pelo menos 3 conjuntos distintos e da forma que conseguirem organizar na página. O layout de vocês deve apresentar, obrigatoriamente os seguinte requisitos:

  • Três seções visivelmente distintas (header, barra lateral, conteúdo central)
  • Toda seção deve trazer pelo menos três elementos dentro que devem ser organizados como componentes usando a tag
    como a tag inicial
  • Deve haver algum formulário com pelo menos 2 campos e um botão (login, subscribe, pesquisa, etc)
  • Não há obrigação de ficarem completamente posicionados e nem certinho como nas imagens.
